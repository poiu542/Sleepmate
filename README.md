# :clipboard: 소개
슬립메이트(SleepMate) 는 유저들의 수면 자세와 생체정보를 이용해 유저들의 수면을 분석하고 올바른 수면습관을 추천해주는 서비스입니다. RTSP 프로토콜 지원 카메라, 워치의 심박수와 조도센서 및 동작센서에서 얻어온 정보를 활용하여 통합적인 수면 진단을 제공합니다.
